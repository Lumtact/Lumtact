# 工程碎片池

> 本文件存放从白皮书 v9.0.0 解耦出的具体实现细节。

---

## 1. 颜色基因

*   **涟漪层色相**：#2B8CBE (LCH: 50, 40, 260)
*   **背景色 (浅色)**：#F8F9FA
*   **背景色 (深灰)**：#121212
*   **背景色 (纯黑)**：#000000
*   **微光层级 (纯黑模式)**：#101010 (约2.5%差异)
*   **遮罩终点色 (暗黑模式)**：oklch(from var(--bg-dark) calc(l + 0.08) c h)
*   **正文 (浅色)**：#1A1A1A
*   **正文 (深色)**：#E0E0E0

---

## 2. 涟漪层透明度

| 状态 | Alpha 值 | 备注 |
|---|---|---|
| 强调/聚焦 | 0.40 | 跨越韦伯阈值 |
| 选中/按下 | 0.30 | 跨越韦伯阈值 |
| 活跃/悬停 | 0.15 | 浅色/深色跨越；纯黑需验证 |
| 分割/引导 | 0.06 | 浅色/深色跨越；纯黑不跨越 |
| 环境/微光 | 0.02 - 0.04 | 仅浅色/深色跨越；纯黑视觉失效 |

---

## 3. 排版与空间

*   **基准字号**：16px (Body)
*   **最小字号**：12px (Caption)
*   **行高基准**：1.5 (西文) / 1.8 (中文)
*   **网格系统**：4px 基准网格。
*   **热区下限**：44px * 44px (手指指尖接触面积)
*   **屏幕边缘豁免热区**：32px
*   **默认圆角**：8px - 12px (移动端), 4px - 8px (桌面端)
*   **最小分割线宽**：2px (防止 Pentile OLED 彩边)
*   **遮罩 Alpha 范围**：0 -> 0.10-0.15

---

## 4. 时间与动效

*   **因果感知下限**：100ms
*   **注意力切换舒适区**：200-350ms
*   **引导动画上限**：400-600ms
*   **交互流转消散时间**：300-500ms
*   **等待焦虑阈值**：1s
*   **持续注意力周期**：10-20分钟
*   **未读信息衰减周期**：24小时 (工程约定)

---

## 5. 技术实现

*   **状态属性**：`data-state` (逻辑层裁决)
*   **媒体查询**：`prefers-reduced-motion` (响应无动效偏好)
*   **色彩函数**：`oklch`, `rgba`
*   **渲染混合**：Alpha 合成需避免非预期混合
